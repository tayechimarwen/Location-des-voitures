Location des Voitures

Application de gestion de location de voitures développée avec Python et SQLAlchemy.

## Technologies Utilisées
- Python
- SQLAlchemy (ORM)
- SQLite (Base de données)
- Flask/FastAPI (si تستخدم framework)

## Fonctionnalités
- Gestion des voitures
- Gestion des clients
- Gestion des locations
- Suivi des retours
